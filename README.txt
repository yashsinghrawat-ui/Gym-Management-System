
# Gym Management System

## Technologies Used
- Java Swing
- MySQL
- Python
- JDBC

## Features
- Add gym members
- View members
- Store data in MySQL
- Generate reports using Python

## Setup Steps

### 1. Install Requirements
- Java JDK
- MySQL Server
- MySQL Connector JAR
- Python

### 2. Create Database
Run the SQL file:
database.sql

### 3. Compile Java Program
javac GymManagementSystem.java

### 4. Run Java Program
java GymManagementSystem

### 5. Run Python Report
python report.py

## Important
Change MySQL password in:
- GymManagementSystem.java
- report.py
